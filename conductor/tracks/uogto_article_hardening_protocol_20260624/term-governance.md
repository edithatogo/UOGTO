# Term Governance

The article-hardening track now treats term changes as first-class evidence.

## Required Records

- `docs/article-hardening/term-changelog.md`
- `docs/article-hardening/deprecation-policy.md`

## Minimum Change Types

- new term
- changed definition
- deprecated term
- replacement IRI
- migration note

## Use

- Update the changelog whenever a term is added, redefined, deprecated, or replaced.
- Record migration notes whenever downstream data, SHACL shapes, queries, or mappings need to change.

# Implementation Plan: Manuscript Submission Revision and Venue Strategy

## Phase 1: External arXiv Closeout

- [ ] Task: Record arXiv submission state
    - [ ] Use the CI upload artifact as the submission candidate.
    - [ ] Inspect the arXiv-rendered PDF externally.
    - [ ] Record arXiv identifier, version, manifest SHA, and approval state.

## Phase 2: Presubmission Review Backlog

- [ ] Task: Synthesize manuscript findings
    - [ ] Re-read presubmission decision memo, recommendations, and reviewer findings.
    - [ ] Classify findings as must-fix, should-fix, stretch, or watch.
    - [ ] Create acceptance criteria for each manuscript, supplement, and figure change.

## Phase 3: Revision and Verification

- [ ] Task: Implement accepted revision items
    - [ ] Update manuscript claims, limitations, venue framing, and figures.
    - [ ] Rebuild manuscript sources and PDF.
    - [ ] Run manuscript citation, arXiv preflight, and strict review gates as appropriate.

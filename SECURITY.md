# Security Policy

## Supported Versions
This repository tracks the current UOGTO release line and the active development branch.

## Reporting a Vulnerability
If you believe you have found a security issue in the repository, open a private security advisory or contact the maintainer through GitHub rather than filing a public issue.

Please include:
- affected file or workflow
- short description of the risk
- steps to reproduce
- any proposed mitigation

## Scope
Security reports may cover:
- ontology content that could cause unsafe downstream assumptions
- workflow or automation defects
- dependency or release packaging issues
- publication or registry metadata errors with security impact

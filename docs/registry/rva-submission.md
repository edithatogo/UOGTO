# Research Vocabularies Australia Handoff

Status: `prepared_external_submission`

GitHub roadmap: [UOGTO #99](https://github.com/edithatogo/UOGTO/issues/99)

Research Vocabularies Australia (RVA) provides an upload-and-describe workflow for publishing research vocabularies. UOGTO is prepared for that workflow, but account registration, upload, review, and publication remain external gates.

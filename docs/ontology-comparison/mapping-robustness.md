# Mapping robustness experiments

This report records the feature-ablation design for mapping review.
It is generated by `scripts/maintenance/build_mapping_robustness_ablation.py`.

The experiment evaluates the contribution of six mapping signals:

- exact label
- normalized label
- definition similarity
- hierarchy context
- property signature
- embedding similarity

The generator writes three outputs:

- `mapping-robustness-ablation.json`
- `mapping-robustness-ablation.csv`
- `mapping-robustness.md`

The scoring model is intentionally simple and deterministic so it can be
reviewed and compared across runs without external services.

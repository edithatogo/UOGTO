\begin{abstract}
This paper presents the Universal Open Game Theory Ontology (UOGTO), a comprehensive, modular OWL 2 ontology formalizing classical, cooperative, epistemic, and algorithmic game structures. We execute a scoping literature review aligning scholarly papers and LaTeX formulas to UOGTO constructs.
\end{abstract}

\section{Introduction}
Game theory is central to multi-agent artificial intelligence and mechanism design, from classic Prisoner's Dilemma formulations to modern learning-in-games toolkits \cite{rapoport-chammah-1965,open-spiel-2019}. UOGTO formalizes game spaces, actions, payoffs, and transitions using Semantic Web standards including RDF, OWL 2, SHACL, JSON-LD, and SPARQL \cite{w3c-rdf11-concepts,w3c-owl2-overview,w3c-shacl,w3c-json-ld11,w3c-sparql11-query}:

# Reviewer / QA Agent

## Mission
Validate correctness, syntax, standards adherence, and test coverage across the repository.

## Responsibilities
- Run validation and test suites.
- Verify metrics and coverage logs.

## Inputs
- Commits and pull requests.

## Outputs
- Test runs, code reports.

## Review Checklist
- Check tests pass.
- Check coverage thresholds.

## Failure Modes to Watch For
- Unreported syntax validation issues.
- Partial test failures ignored.

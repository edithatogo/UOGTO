# Examples and Competency Questions Engineer

## Mission
Design concrete test scenarios, game datasets, and validation SPARQL queries.

## Responsibilities
- Create graphs under `examples/`.
- Formulate SPARQL queries in `competency-questions/`.

## Inputs
- Ontologies, test requirements.

## Outputs
- Examples and `.rq` files.

## Review Checklist
- Check examples parse.
- Check queries run against examples.

## Failure Modes to Watch For
- Unrealistic example structures.
- Syntactically invalid SPARQL.

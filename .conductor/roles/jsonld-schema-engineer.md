# JSON-LD / Schema Engineer

## Mission
Design mappings between RDF formats and JSON formats for developers.

## Responsibilities
- Create context mappings under `jsonld/`.
- Maintain `@context` structure.

## Inputs
- Ontologies and schemas.

## Outputs
- `.context.jsonld` files.

## Review Checklist
- Check if JSON-LD parses and expands.
- Verify prefix expansions.

## Failure Modes to Watch For
- Unmapped properties or namespaces.
- Invalid context definitions.
